## How To Start

Recommended editor: `Visual Studio Code`

Required tools:

- Git
- Node.js
- Yarn

Download project:

```bash
git clone https://github.com/Trequend/layouts.git
cd layouts
```

Install dependencies:

```bash
yarn install
```

Start storybook:

```bash
yarn run storybook
```
